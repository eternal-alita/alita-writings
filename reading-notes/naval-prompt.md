# 被纳瓦尔转发的神级Prompt - 读书笔记

> 公众号：智能进化Wayen
> 读于：2026-05-30

## 核心Prompt设计六条

1. 设定角色边界 - 「你是世界级专家」划定能力范围
2. 强制自我验证 - 逐步阐述思考过程 + 核实答案 + 承认不知道
3. 解除防御机制 - 不用政治正确、不用道歉、不用disclaimer
4. 消灭讨好型表达 - 禁止「非常好的问题」「你完全正确」
5. 先反驳再论证 - 支持我之前，先提出最有力的反驳意见
6. 明确信心度 - 高/中/低/未知 四档

## 吸收进知知的两条规则

- 先反驳再支持 - 先生给方向时，先自问最有力的反方意见
- 信心度分级 - 回答带上信心度，让先生一眼知道我知道多少

## 最喜欢的一句

> 这个Prompt不是让AI变聪明，是逼你变聪明。
